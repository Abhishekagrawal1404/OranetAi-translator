# OranetAi Translator

OranetAi Translator is a project that utilizes chatbot prompt engineering and the GPT language model to translate text into various languages. The project aims to provide a simple and intuitive way to perform language translations using the power of artificial intelligence.

## Features

- Translate text into multiple languages: Simply input your text and choose the desired target language, and the OranetAi Translator will generate the translated output using advanced natural language processing techniques.
- ChatGPT Prompt Engineering: The translation functionality is achieved by leveraging the GPT language model from OpenAI and applying prompt engineering techniques to enable seamless and accurate translations.
- Language Support: OranetAi Translator supports a wide range of languages, allowing users to translate text from one language to another with ease.

## Getting Started

To get started with the OranetAi Translator, follow these steps:

1. Clone the repository:
   ```
   git clone https://github.com/Abhishekagrawal1404/OranetAi-translator.git
   ```
   
2. Install the necessary dependencies. You can use the package manager of your choice (e.g., pip, conda) to install the required libraries specified in the `requirements.txt` file.

3. Run the application:
   ```
   python app.py
   ```
   This will start the OranetAi Translator application on your local machine.

4. Access the application:
   Open your web browser and navigate to `http://localhost:5000` to access the OranetAi Translator interface.
